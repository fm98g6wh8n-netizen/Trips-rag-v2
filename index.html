TradeGuide v2가 출처 기반의 검증된 답변을 제공하는 방법

RAG (Retrieval-Augmented Generation) 구조

TradeGuide v2는 답변을 지어내지 않습니다. 내장된 법규 문서를 먼저 검색(Retrieval)한 뒤, 검색 결과만을 근거로 답변을 구성합니다. 모든 답변에는 출처 조항과 TF-IDF 검색 점수가 함께 표시되어, 사용자가 신뢰도를 직접 확인할 수 있습니다.

🗄️
지식 베이스

WTO TRIPS 4개 조항 + 한미 통상법 3개 항목을 내부 데이터로 색인.

🔍
TF-IDF 검색

한국어 형태소 분리·동의어 확장 후 코사인 유사도로 Top-3 추출, 도메인 가중치 적용.

📝
룰 기반 답변

검색된 원문을 룰 템플릿으로 종합. LLM 환각 없이 출처에 고정된 답변.

2차 버전의 핵심 — Agentic + 자체 검증

🧩
질문 분해

조항 번호가 여럿이거나 "비교/차이"가 포함되면 하위 질문으로 분해해 각각 검색합니다.

🔗
다중 검색·통합

하위 질문마다 별도 Top-3 검색 후 결과를 병합해 통합 답변을 생성합니다.

🛡️
3유형 자체 검증

개념 대체·도메인 혼동·기간 혼동 오류를 검사해 신뢰도 ★★★ / ★★를 표시합니다.

  • 전처리: 한국어 조사·어미 단순 분리, 영문 토큰화, 동의어 사전 확장
  • 외부 의존성: 없음 (CDN·빌드 도구·백엔드 서버·API 키 모두 미사용)

시스템 한계 안내. 본 시스템은 학생부종합전형 동아리 활동 산출물로, TF-IDF 기반 정적 검색과 룰 기반 답변 템플릿으로 구성됩니다. LLM 추론 능력은 없으며, 내장된 WTO TRIPS 4개 조항(제27·31·35–38·39)과 한미 통상법 3개 항목에 한해서만 응답합니다. 검증 레이어의 "신뢰도 ★★★"은 3개 휴리스틱 규칙 통과를 의미할 뿐 법적 정확성을 보증하지 않습니다. 관세 시나리오 계산기는 단순 산술 모델이며 실제 거래에는 운임·부가세·원산지 규정 등이 추가로 반영되어야 합니다. 실제 법률·통상 판단에는 원문 및 전문가 자문이 필요합니다.
